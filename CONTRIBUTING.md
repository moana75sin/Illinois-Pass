# Contributing to Illinois Pass

Thank you for your interest in contributing to Illinois Pass. Contributions from developers, designers, testers, and community members help improve this project.

## Contribution Guidelines

Before contributing, please follow these guidelines:

- Write clean and readable code.
- Follow existing project formatting styles.
- Keep components reusable and organized.
- Add comments when complex logic is introduced.
- Test your changes before submitting.
- Maintain respectful communication.

All contributions should improve the reliability, usability, or performance of Illinois Pass.
